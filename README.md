# 📸 찍자 ZZikZZa

> 친구에게 사진을 부탁했지만 원하는 대로 나오지 않아 아쉬웠던 경험 있으신가요?  
찍어줄 때마다 상대방이 마음에 들어 하지 않아 여러 번 재촬영을 한 적도 있으신가요?  
`찍자 ZZikZZa`가 실시간 피드백과 직관적인 가이드로 사진 촬영을 원활하게 도와드릴게요!

![ZZikZZa 로고](./Documents/Attachments/ZZikZZa.png)

찍는 사람의 촬영 화면을 찍히는 사람이 자신의 휴대폰에서 실시간으로 함께 보고 시각적 가이드를 공유해, 더 원활하고 정확한 소통을 돕는 카메라 앱

## :pushpin: 주요 기능

- 실시간 카메라 프리뷰 스트리밍 기능
- 기본 카메라 기능을 최대한 담은 촬영 기능
- 촬영을 위한 쌍방향 소통을 돕는 가이딩 기능

## :sparkles: 기술

`Swift`, `SwiftUI`, `UIKit`, `AVFoundation`, `VideoToolBox`, `Wi-Fi Aware`
