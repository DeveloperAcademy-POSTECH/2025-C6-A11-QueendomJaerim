# File Comments

## 주석과 파일 헤더

- Swift 파일 헤더는 Xcode 기본 스타일을 유지한다.
- `Created by`에는 LLM 이름이나 도구 이름을 쓰지 않는다.
- 새 파일 생성자는 실제 사용자 이름 또는 해당 작업을 수행하는 사람의 이름을 사용한다.
- 불필요한 설명 주석은 추가하지 않는다. 코드만으로 의도가 드러나지 않는 복잡한 분기나 정책에만 짧게 남긴다.

예시: [QueenCam/QueenCam/Presentation/Views/Common/Settings/Main/SettingsMainView.swift](../../QueenCam/QueenCam/Presentation/Views/Common/Settings/Main/SettingsMainView.swift)

```swift
//
//  SettingsView.swift
//  QueenCam
//
//  Created by 임영택 on 2/14/26.
//
```
